## STACK

- NestJS
- Express
- Typescript
- MongoDB
- Docker

## Installation

```bash
$ npm i -g @nestjs/cli
$ npm install
```

## Running the app

Upload Docker Database

```bash
$ docker-compose up -d
```

Run Nest project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
